# Desarrollo-de-una-Herramienta-de-Verificaci-n-de-Seguridad-en-Redes-WiFi-Gratuitas

El propósito de este taller es desarrollar una aplicación en Python que permita auditar de
manera segura la conexión a redes WiFi gratuitas, verificando que los sitios web accedidos
desde estas redes no sean maliciosos. La herramienta se centrará en la comprobación de
posibles ataques de phishing mediante la validación de los DNS reales de un conjunto de
sitios web previamente definidos, comparando las direcciones IP de resolución legítima con
las obtenidas en la conexión activa.
